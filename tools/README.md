# Tools - Business Context AI Template

Helper tools and checklists to make filling and maintaining your business context easier.

---

## üìã Available Tools

### [Completion Checklist](completion-checklist.md)
**Track your progress filling out the template**

- Step-by-step checklist for all 22 sections
- Priority levels (Must Have, Recommended, Good to Have)
- Time estimates per section
- Progress tracking
- Quality verification checklist

**Use when**: Filling out the template for the first time or expanding existing context

**Time**: Ongoing reference throughout the filling process

---

### [Validation Checklist](validation-checklist.md)
**Ensure your context is complete and AI-ready**

- Completeness verification
- Quality checks (specificity, explanation, accuracy)
- Content quality assessment
- Security and privacy check
- Stakeholder validation tracking
- AI readiness testing

**Use when**: Before publishing your context or after major updates

**Time**: 30-60 minutes for full validation

---

### [Update Template](update-template.md)
**Structured approach to updating your context**

- Update trigger identification
- Impact scope determination
- Change documentation
- Version control guidance
- Common update scenarios with checklists
- Rollback procedures

**Use when**: Making changes to existing business context

**Time**: Varies by update scope (30 min - 6 hours)

---

### [Section Priority Guide](section-priority-guide.md)
**Which sections to fill first based on your needs**

- Multiple paths: Quick Start, Startup, Scale-up, Enterprise
- Problem-specific priorities
- Time budgets by role
- Decision matrix
- Quick assessment questions

**Use when**: Planning which sections to fill, prioritizing work

**Time**: 10-15 minutes to read and decide

---

## üéØ Recommended Workflow

### 1. Planning (Use Section Priority Guide)
- Assess your situation
- Choose appropriate path
- Understand time commitment
- Identify priorities

### 2. Filling (Use Completion Checklist)
- Work through priority 1 sections first
- Track progress as you go
- Check off completed items
- Note sections to skip

### 3. Validation (Use Validation Checklist)
- Run completeness check
- Verify quality
- Get stakeholder reviews
- Test with AI tool

### 4. Maintenance (Use Update Template)
- Update when things change
- Follow structured process
- Document changes
- Version appropriately

## üí° Quick Start Workflow

**If you need context TODAY** (2-4 hours):

1. **Read** [Section Priority Guide](section-priority-guide.md) - Quick Start Path (10 min)
2. **Fill** Priority 1 sections only (2-3 hours)
3. **Skip** formal validation for now
4. **Use** in your next AI project (immediate)
5. **Note** what questions AI still asks
6. **Expand** based on gaps (later)

## üÜò Common Questions

### "Do I need to use all these tools?"

No! Use what helps you:
- **Minimum**: Section Priority Guide (to know where to start)
- **Recommended**: + Completion Checklist (to track progress)
- **Comprehensive**: + Validation and Update tools

### "Can I customize these tools?"

Absolutely! These are starting points:
- Add company-specific sections
- Remove sections that don't apply
- Adjust time estimates for your context
- Add industry-specific checks

### "How do I track progress digitally?"

Options:
- Convert to GitHub Issues
- Use project management tool (Jira, Asana, etc.)
- Create spreadsheet
- Use note-taking app (Notion, Obsidian)

### "The checklists are too detailed. Can I simplify?"

Yes! Create your own simplified version:
- Focus on Priority 1 sections only
- Remove granular checks
- Just track major sections completed
- Add detail as you need it
